# TMM-2024-TSSR

This repo contains the MATLAB code of our TMM'2024 paper: [Tensorized scaled simplex representation for multi-view clustering (TSSR)](https://doi.org/10.1109/TMM.2024.3355649).

## Requirements

- MATLAB R2018b

## Run

```
run_TSSR.m
```
